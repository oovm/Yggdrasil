# WHITESPACE
Remove from rendering result

- inline mode

```notedown
\comment: something will not shown
```

- text mode

```notedown
\comment[something will not shown]
```

# NEWLINE
Embed images in documents
- smart link

```notedown
[some/path.png]
```

- inline mode

```notedown
\img: something will not shown`
```

- text mode

```notedown
\img[some tips not shown]
```
